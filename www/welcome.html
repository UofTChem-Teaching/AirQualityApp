Welcome

As part of Experiment 1, you've already analyzed a small subset of the ECCC NAPS data. To see the rest, we've created this website where you're able to explore all of the O3, NO2 measurements recorded by the NAPS network across Canada in 2018.

How to navigate the website

This website is broken into five parts: the side panel and the four tabs at the top of the page.

  • The side panel is where you can see find all available NAPS stations (click on a marker for station details); a dropdown menu to select the station and date range you'd like to explore and options to modify the plots (see Plots tab).
  • Welcome tab, what you're reading right now.
  • Plots tab, which contains the time series and O3 vs. NO2 correlation plots. The side panel options modify these plots.
  • Summary stats tab which displays some simple summary stats of the selected data; this is the same data shown in the Plots tab
  • Notes tab with sources on data.
